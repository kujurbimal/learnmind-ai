## ⚡ How the Automation Works

```
TRIGGER (one of three ways)
  1. Every Monday 9am UTC (scheduled)
  2. Edit content-config.json and push
  3. GitHub UI → Actions → Run workflow
          │
          ▼
GitHub Actions Runner
  └── scripts/generate.js
       ├── Calls Claude API for each of 9 sections
       ├── Saves JSON to content/*.json
       └── Builds populated index.html
          │
          ▼
Git commit pushed to main
  "🤖 AI pipeline: regenerate content [timestamp]"
          │
          ▼
GitHub Pages auto-deploys (~60 seconds)
  yourname.github.io/learnmind-ai ✅
```

## 🚀 Setup Guide (one-time, ~10 minutes)

### Step 1 — Fork or clone

```bash
git clone https://github.com/YOUR-USERNAME/learnmind-ai.git
cd learnmind-ai
```

### Step 2 — Add your Anthropic API key as a GitHub Secret

1. Go to your repo → **Settings → Secrets and variables → Actions**
2. Click **New repository secret**
3. Name: `ANTHROPIC_API_KEY`
4. Value: your key from [console.anthropic.com](https://console.anthropic.com)
5. Click **Add secret**

> ⚠️ Never put your API key in any file — only in GitHub Secrets.

### Step 3 — Enable GitHub Pages

1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main`, Folder: `/ (root)`
4. Click **Save**

### Step 4 — Trigger the first run

**Option A — GitHub UI:**
1. Go to **Actions → AI Content Pipeline → Run workflow**
2. Fill in Brand, Audience, Tone (or leave defaults)
3. Click **Run workflow** ✅

**Option B — push config:**
```bash
# Edit content-config.json then push to auto-trigger
git add content-config.json
git commit -m "Configure pipeline"
git push
```

The pipeline takes ~2 minutes. Your live site will have fully AI-generated content.

## ⚙️ Configuring the Pipeline

Edit **`content-config.json`**:

```json
{
  "brand":    "LearnMind.AI",
  "audience": "K-12 students and parents",
  "tone":     "motivational and urgent",
  "sections": "all"
}
```

| Field | Options |
|-------|---------|
| `brand` | Your product name |
| `audience` | Who you're targeting |
| `tone` | `motivational and urgent` / `friendly and approachable` / `professional and authoritative` / `playful and energetic` |
| `sections` | `all` or comma-separated IDs: `hero,testimonials,cta` |

**Partial regeneration** — only refresh specific sections (uses cached JSON for the rest):
```json
{ "sections": "hero,testimonials,cta" }
```

## 🕐 Changing the Schedule

Edit `.github/workflows/ai-pipeline.yml`:

```yaml
schedule:
  - cron: '0 9 * * 1'    # Every Monday 9am UTC (default)
  # - cron: '0 9 * * *'  # Every day
  # - cron: '0 9 1 * *'  # First of every month
```

Remove the `schedule:` block entirely to disable auto-runs.

## 🛠 Running Locally

```bash
cd scripts
npm install

# Full run
ANTHROPIC_API_KEY=sk-... node generate.js

# Custom brand
ANTHROPIC_API_KEY=sk-... BRAND="MySchool.AI" TONE="friendly and approachable" node generate.js

# Regenerate only specific sections
ANTHROPIC_API_KEY=sk-... SECTIONS=hero,testimonials node generate.js
```
